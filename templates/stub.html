Team Project Ideas:

Our team project will be similar to the clock app on iphone where there are many functions. These functions include showing the time through the use of a digital clock, timer, stopwatch, and showing different times in different time zones. All of these functions essentially serve the same purpose in different ways: To allow the user to keep track of time.

Team Project Plan:

Here is our plan for how we will complete each component of the rubric:

Row 1: Program Purpose and Function The purpose of the program is to allow the user to use the many features available relating to time.

Row 2: Data Abstraction The data in the program will be stored mainly in dictionaries. Dictionaries can be used to store times for different time zones.

Row 4: Procedural Abstraction We will make it so that it separates the timezones for each part of the world. We will also separate this from the other features in our clock app.

Row 5: Algorithm Implementation We will be use APIs, sequencing, selecting, and iterating in order to make the program compile the clock app.

Row 6: Testing This program will need to be tested a lot to make sure all the features work both individually and together.
